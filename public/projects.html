Shared Memory Chat Application (C / Linux Systems)

  • Overview: A real-time, multi-process chat application where different terminal instances communicate via shared memory segments.
  • Technologies: C, POSIX Shared Memory, Semaphores, Pthreads, Linux API, GCC, Make
  • Technical Highlights:
    • Implemented POSIX Shared Memory (shm_open) and Semaphores to manage inter-process communication (IPC) and prevent race conditions.
    • Utilized Multithreading (Pthreads) to separate message sending (Main Thread) from receiving (Reader Thread), ensuring non-blocking I/O.
    • Designed a Circular Buffer mechanism to handle message persistence and synchronization across multiple active users.
